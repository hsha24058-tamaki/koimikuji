おみくじアプリ（恋愛バージョン）

■使い方
ボタンを押すと、恋愛運がランダムに表示されます。
恋みくじアプリ (Love Fortune App)ボタンをクリックするだけで今日の恋愛運を占うことができる、シンプルなWebアプリケーションです。
構成ファイルこのプロジェクトは以下の3つのファイルで構成されています。
ファイル名役割index.htmlアプリの構造（タイトル、ボタン、表示エリア）を定義style.cssアプリのデザイン（色、配置、フォント）を指定script.jsおみくじの抽選ロジックと画面更新を制御使い方すべてのファイルを同じフォルダ（ディレクトリ）に保存します。index.html をブラウザで開きます。
「おみくじを引く」ボタンをクリックすると、ランダムに運勢が表示されます。
